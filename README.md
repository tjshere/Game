# Game (working title)

A single-player browser game with old-school MMO bones. Plain HTML/CSS/JS,
Canvas 2D, zero dependencies. Open `index.html` via any static server.

- Design spec: `docs/specs/2026-07-08-game-v1-design.md`
- Run tests: `npm test` (Node 20+, uses built-in `node --test`)
- Drop rates live in `js/drops.js`; the world map is text in `js/world.js`.
